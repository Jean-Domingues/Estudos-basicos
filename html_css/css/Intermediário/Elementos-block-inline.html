Elementos inline, block, inline-block

Elementos inline

Tags como, a, img, span, são containers que ficam dispostos um ao lado do outro, e ocupam somente o espaço do tamanho do seu conteúdo.

Elementos Block

Tags como, h1, p, table, são elementos que ocupam todo o espaço da tela, e ficam dispostos um abaixo do outro.

Elementos inline-block

Tags como o display, e inline-block, tem a largura definida baseada no seu conteúdo, pórem os itens ficam um abaixo do outro quando não há mais espaço.
