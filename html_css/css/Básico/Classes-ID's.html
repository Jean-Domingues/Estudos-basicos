Classes

Uma classe é utilizada para definir propriedades a um grupo específico que você queira, como por exemplo mudar a cor de algum páragrafo para azul, assim criamos uma classe, e dentro dela atribuimos a cor azul. Para criar uma classe, precisamos inicia-la com um ponto, por exemplo: .branco. Esta classe pode ser atribuida a qualquer tag, basta chama-la, utilizando o comando class="nome da classe, sem o ponto".

ID's

Ao contrário das classes, os ID's só podem ser utilzados por um único elemento específico, pois se tivermos mais de um por página, a recomendação é que utilizamos classes. Os ID's seguem o mesmo padrão de declaração e função de uma classe, sendo diferente apenas na troca do ponto pelo jogo da velha na declaração, e na chamada, sendo realizada pelo comando, id="nome do id".
