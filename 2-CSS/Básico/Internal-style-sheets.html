Internal style sheets

O internal style sheets se diz referente ao css dentro do html, porém em uma ala separada, que seria a tag style, que vai na "cabeça" do html. Ele serve para controlarmos as propriedades de diversas tags ao mesmo tempo (seletores), poupando o trabalho de ter que por um comando em cada uma. Como visto neste páragrafo, o inline style, sobrepõe o internal style.
